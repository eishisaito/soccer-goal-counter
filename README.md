# サッカーゴールかぞえ

サッカーゴールかぞえは、サッカーのゴールを数えるサービスです。ユーザーは、ゴールボタンを押してゴールを数えることができます。このアプリは、Web技術（HTML、CSS、JavaScript）で構築されており、PWA（Progressive Web App）としても利用できます。


## 主な機能

- ゴールをカウントする機能
- ゴール後にアニメーション
- ポイントの状態がlocalStorageに保存される
- PWA対応（ホーム画面に追加可能）
